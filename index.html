📖 Setup Guide

🚀 Get Started in 5 Minutes — Free
1
Create a free EmailJS account

Go to emailjs.com → Sign Up → Verify your email

2
Add an Email Service

Dashboard → Email Services → Add New Service → Connect Gmail/Outlook → Copy the Service ID

3
Create an Email Template

Email Templates → Create Template → Set To field as {{to_email}} → Add {{subject}}, {{message}}, {{from_name}} in the body → Save → Copy Template ID

4
Get Your Public Key

Profile icon → Account → General tab → Copy Public Key

5
Paste into Email Config

Go to ⚙️ Email Config → Paste all 3 keys → Click Save → Switch to 🟢 Real Send mode

⚠️ Important Notes
  • Free plan: 200 emails/month — perfect for testing & small businesses
  • Always set {{to_email}} in the EmailJS Template To field — not just the body
  • Use a delay of 2–3 seconds between emails to avoid spam filters
  • Your config is saved locally in your browser — stays private
  • For unlimited sending, upgrade to EmailJS paid plan or use SendGrid SMTP
